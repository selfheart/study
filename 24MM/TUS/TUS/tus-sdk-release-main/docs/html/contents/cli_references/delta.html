4.5. Delta Subcommand¶

$ tus delta -h
Usage: tus delta [OPTIONS] [COMMAND]

Description:
  TUS-SDK delta update command

Options:
  -h         print help message

Commands:
    diff       create differential data
    patch      apply patch

4.5.1. Create differential data¶

tus delta diff command creates a delta file from a source file and a destination file.

Detail usage is following:

$ tus delta diff -h
Usage: tus delta diff [OPTIONS] SRC_FILE DST_FILE DELTA_FILE

Description:
  Create DELTA_FILE from SRC_FILE and DST_FILE.

Options:
  -h                      print help message
  --algorithm ALGORITHM   specify delta algorithm (default:bsdiff)

--algorithm specifies the delta algorithm. --algorithm is optional. If not specified, the delta algorithm will be bsdiff. Currently only bsdiff is supported as --algorithm.

4.5.2. Apply patch¶

tus delta patch command applies a delta file to a source file, to create a patched file.

Detail usage is following:

$ tus delta patch -h
Usage: tus delta patch [OPTIONS] SRC_FILE PATCHED_FILE DELTA_FILE

Description:
  Apply DELTA_FILE to SRC_FILE, to create PATCHED_FILE.

Options:
  -h                      print help message
  --algorithm ALGORITHM   specify delta algorithm (default:bsdiff)

--algorithm specifies the delta algorithm. --algorithm is optional. If not specified, the delta algorithm will be bsdiff. Currently only bsdiff is supported as --algorithm.
