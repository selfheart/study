4.3. PDP SubcommandÂ¶

In this document, PDP stands for Package Delivery Platform. For example, EMOOTA SDP v3.0 can be regarded one of the PDPs.

PDP subcommand handles PDP operations like uploading TUP and registering campaign.

$ tus pdp -h
Usage: tus pdp [COMMAND] [OPTIONS]

Description:
  TUS-SDK PDP related operation command

Options:
  -h    print help message

Commands:
    sdpv3       operate EMOOTA SDP v3

Currently only EMOOTA SDP v3.0 is supported as PDP. Each PDP will have common operations like uploading TUP and PDP specific operations.
